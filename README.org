* Overview

This project demonstrates instance segmentation using PyTorch and the Mask R-CNN model with a ResNet-50 FPN backbone. It trains on the Penn-Fudan Pedestrian dataset to detect and segment pedestrians in images.

* Features

- Pre-trained Mask R-CNN model fine-tuned for pedestrian detection
- Instance-level segmentation with bounding boxes and masks
- Data augmentation with random horizontal flipping
- COCO-style evaluation metrics

* Requirements

- Python 3.x
- PyTorch
- torchvision
- NumPy
- PIL

* Installation

Clone the repository:

#+begin_src shell
git clone https://github.com/conao3/object-segment.py.git
cd object-segment.py
#+end_src

* Usage

1. Download and extract the Penn-Fudan dataset:

#+begin_src shell
wget https://www.cis.upenn.edu/~jshi/ped_html/PennFudanPed.zip
unzip PennFudanPed.zip
#+end_src

2. Run the training script:

#+begin_src shell
./object-segment.py
#+end_src

The script will train for 10 epochs and display segmentation results on a test image.
